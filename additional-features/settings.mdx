## Overview

Access settings from **Settings** in the left sidebar. Settings are organized into four tabs: **Main**, **Tags**, **RPC**, and **Browser**.

## Main

<Frame><img src="/images/atmos_ui/settings/atmos_settings_main.png" alt="Settings — Main tab" /></Frame>

| Setting | Description |
|---|---|
| **API key** | Change the API key linked to your account |
| **Master password** | Change the master password used to encrypt all application data |
| **PIN code** | Set or change the PIN used to confirm sensitive actions (transfers, deletions) |
| **Language** | Interface language |

### Export

Export your data as encrypted ZIP archives. Three categories available:

| Category | Contents |
|---|---|
| **Wallets** | Wallet data including tags and personalization |
| **Accounts** | CEX account data including tags and personalization |
| **Software settings** | RPC configuration, extensions, and other app settings |

### Import

Drag and drop or click **Select** to import a previously exported encrypted ZIP archive.

## Tags

Tags in Oxygen Atmos are **unified across wallets, browser profiles, and exchange accounts**. A tag created anywhere is available everywhere.

<Frame><img src="/images/atmos_ui/settings/atmos_settings_tags.png" alt="Tags management table" /></Frame>

### Tags table

| Column | Description |
|---|---|
| **Name** | Tag name with color indicator |
| **Count** | Total items using this tag |
| **Browsers** | Checkbox — tag is used on browser profiles |
| **Wallets** | Checkbox — tag is used on wallets |
| **Accounts** | Checkbox — tag is used on exchange accounts |
| **Extensions** | Number of profiles with extensions assigned to this tag |

You can filter the table with **Colors**, **Search**, and **Show only unused**.

### Create a tag

Click **Add tag**, enter a **name** and choose a **color**.

### Assign tags

You can assign tags in several places:

- **Wallet manager** — during import or when editing a wallet
- **Browser profiles** — during creation, bulk creation, or when editing a profile
- **Exchange accounts** — when adding or editing an account

You can apply multiple tags to a single item.

### Edit and delete tags

- Click the **edit icon** to change a tag's name or color. Changes apply everywhere the tag is used.
- Select one or more tags and click **Delete** to remove them.

<Warning>
  Deleted tags are automatically removed from all wallets, profiles, and accounts where they were assigned.
</Warning>

### Filter by tags

Both the wallet list and the profile list support tag filtering:

1. Click **Tags** above the list
2. Select one or more tags — only items with those tags appear
3. Click **Reset** to clear the filter

## RPC

Configure RPC endpoints for EVM chains.

| Setting | Description |
|---|---|
| **Use global proxy for all RPCs** | Route all RPC requests through a single proxy |
| **Use proxy from wallet browser profile** | Use each profile's proxy for its RPC requests |
| **Chain list** | Toggle chains on/off, view RPC health status |

Each chain shows a health indicator (e.g. **3/3** = 3 healthy out of 3 total RPCs). You can add up to **5 custom RPC endpoints** per chain. A "Low active RPCs" warning appears when some endpoints are unresponsive.

## Browser

Configure browser fingerprint defaults and manage extensions.

<Frame><img src="/images/atmos_ui/settings/atmos_settings_browser.png" alt="Settings — Browser tab" /></Frame>

### Fingerprint settings

| Setting | Description |
|---|---|
| **Language** | Based on IP address or set manually (e.g. English United States) |
| **App language** | Based on "Language" setting or set independently |
| **Timezone** | Based on IP address |
| **Geolocation** | Ask, Allow, or Block — with "Based on IP address" toggle |
| **WebRTC** | Block unproxied UDP or Public Interface |
| **WebGL Metadata** | Fingerprint, System, or Disable |
| **Noise** | Toggle Canvas, Audio, Rectangles, WebGL |

### Extensions

Extensions are **global** — they apply to all browser profiles.

| Action | Description |
|---|---|
| **Add** | Install from a Chrome Web Store URL or upload a CRX/ZIP file |
| **Update extensions** | Update all installed extensions |
| **Delete** | Remove an extension |

<Info>
  **Rabby Wallet** is a built-in default extension. It includes a security feature: private keys cannot be exported from Rabby manually or by any other means.
</Info>
