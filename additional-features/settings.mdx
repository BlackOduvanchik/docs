## Overview

Access global settings from **Settings** in the sidebar or by pressing `Ctrl+,` (Windows/Linux) or `Cmd+,` (macOS).

## General settings

| Setting | Description | Default |
|---|---|---|
| **Language** | Interface language | English |
| **Theme** | Light or dark mode | System |
| **Auto-launch** | Open Oxygen Atmos on system startup | Disabled |
| **Close behavior** | Minimize to tray or quit on close | Minimize |

## Profile defaults

Configure default values for new profiles:

| Setting | Description |
|---|---|
| **Default OS** | Operating system for new profiles |
| **Default browser** | Browser version for new profiles |
| **Auto-assign proxy** | Automatically assign a proxy from the list |
| **Default folder** | Folder for new profiles |

## Extensions

Browser extensions are managed in **Settings** → **Extensions**.

| Setting | Description |
|---|---|
| **Global extensions** | Extensions loaded automatically in every profile |
| **Import CRX** | Upload a `.crx` file to add an extension |
| **Enable/Disable** | Toggle an extension on or off |
| **Remove** | Uninstall an extension globally |

<Info>
  You can also install extensions per-profile directly from the Chrome Web Store inside a running profile. Per-profile extensions have independent settings and data.
</Info>

<Warning>
  Some extensions can leak your real identity (e.g., extensions that sync with a Google account). Use extensions carefully in antidetect workflows.
</Warning>

## Sync settings

| Setting | Description |
|---|---|
| **Auto-sync** | Sync profiles after closing |
| **Sync frequency** | How often to sync running profiles |
| **Sync on launch** | Download latest data when opening a profile |

## Updates

| Setting | Description |
|---|---|
| **Auto-update** | Automatically install new versions |
| **Update channel** | Stable or Beta |
| **Check for updates** | Manually check for the latest version |

## Data and privacy

| Setting | Description |
|---|---|
| **Clear cache** | Remove cached data for all profiles |
| **Export all data** | Download all your profiles, proxies, and settings as a backup |
| **Delete account** | Permanently delete your Oxygen Atmos account and all data |

<Warning>
  Account deletion is permanent. Export your data first if you want to keep a backup.
</Warning>
