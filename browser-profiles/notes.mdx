## Overview

Profile notes let you attach text annotations to any browser profile. Use notes to track account credentials, campaign details, status updates, or any context you need when working with profiles.

## Adding notes

<Steps>
  <Step title="Open profile details">
    Click on a profile name or select **Edit** from the context menu.

    {/* Screenshot: profile edit button */}
  </Step>
  <Step title="Navigate to Notes tab">
    In the profile editor, click the **Notes** tab.

    {/* Screenshot: Notes tab in profile editor */}
  </Step>
  <Step title="Write your note">
    Enter your text in the notes field. Notes support plain text.

    {/* Screenshot: notes text field with example content */}
  </Step>
  <Step title="Save">
    Click **Save** to store the note with the profile.
  </Step>
</Steps>

## Use cases

| Use case | Example note |
|---|---|
| **Account credentials** | `Email: user@example.com / Pass: ****` |
| **Campaign tracking** | `FB Ads Campaign #42 — started 2024-01-15` |
| **Status** | `Banned on 2024-02-01 — need new proxy` |
| **Instructions** | `Don't change proxy — active session on Amazon` |

<Warning>
  Notes are stored in your Oxygen Atmos account. If you transfer a profile, notes are transferred with it. Do not store sensitive credentials in plain text — use a password manager instead.
</Warning>

## Searching notes

Use the search bar in the main dashboard to search across all profile notes. Type a keyword and results will include profiles where the note contains your search term.
