## Overview

Profile notes let you attach text annotations to any browser profile. Use notes to track account credentials, campaign details, status updates, or any context you need when working with profiles.

## Adding notes

<Steps>
  <Step title="Open profile details">
    Click on actions button and select **Security notes** from the context menu.

  </Step>
  <Step title="Write your note">
    Enter your text in the notes field. Notes support plain text.
  </Step>
  <Step title="Save">
    Click **Save** to store the note with the profile.
  </Step>
</Steps>

## Use cases

| Use case | Example note |
|---|---|
| **Account credentials** | `Email: user@example.com / Pass: ****` |
| **Campaign tracking** | `Lighter Regional Competition — started 2026-01-15` |
| **Status** | `Banned on 2026-02-01 — need new proxy` |
| **Instructions** | `Don't change proxy — active session on Polymarket` |

<Warning>
  Notes are stored locally on your machine. Do not store sensitive credentials in plain text — use a password manager instead.
</Warning>
