## Overview

Mass creation lets you generate dozens or hundreds of browser profiles in a single operation. Each profile gets a unique fingerprint, and you can assign wallets, proxies, and tags in bulk. You can also import profiles from a file.

## Bulk create

<Steps>
  <Step title="Open bulk creation">
    Click **New Profile** and select the bulk creation option.
  </Step>
  <Step title="Set the quantity">
    Enter the number of profiles to create.
  </Step>
  <Step title="Configure options">
    Toggle the options you need:

    - **Wallets** — assign wallets to profiles
    - **Proxies** — assign proxies (one per line, applied sequentially to profiles)

    <Frame><img src="/images/atmos_ui/browser/atmos_browser_bulk_create_1.png" alt="Bulk create — step 1 configuration" /></Frame>
  </Step>
  <Step title="Review and import">
    Review the generated profiles in the table (name, status, tags, proxy). Select a language setting. Click **Import** to create all profiles.
  </Step>
</Steps>

## Proxy format

When assigning proxies during bulk creation or file import, use one of these formats:

```text
protocol://host:port:user:pass
protocol://user:pass@host:port
```

Supported types: **HTTP** and **SOCKS5**. One proxy per line. Proxies are applied sequentially to profiles in the order they appear.

## Tips

- Assign [tags](/additional-features/settings#tags) during creation to group profiles by campaign or purpose
- Use file import for large batches where you need per-profile configuration
