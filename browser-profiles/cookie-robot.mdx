## Overview

Cookie Robot automates the process of "warming up" browser profiles by visiting a list of popular websites. This generates realistic browsing history, cookies, and cache data — making profiles appear more natural to antidetect systems.

## Why warm up profiles

New browser profiles with empty cookies and history can trigger security checks on platforms like Facebook, Google, and Amazon. Cookie Robot solves this by:

- Building a realistic cookie footprint
- Generating browsing history across popular domains
- Creating localStorage and cache entries
- Establishing a believable browsing pattern

## Using Cookie Robot

<Steps>
  <Step title="Select profiles">
    Select one or more profiles from the profile list.

    {/* Screenshot: selected profiles for Cookie Robot */}
  </Step>
  <Step title="Open Cookie Robot">
    Click **Tools** → **Cookie Robot** or right-click and select **Cookie Robot**.

    {/* Screenshot: Cookie Robot menu entry */}
  </Step>
  <Step title="Configure websites">
    Choose which websites to visit:

    - **Default list** — curated set of popular websites (Google, YouTube, Wikipedia, etc.)
    - **Custom list** — add your own URLs

    {/* Screenshot: Cookie Robot website configuration */}
  </Step>
  <Step title="Set parameters">
    Configure the warm-up behavior:

    | Parameter | Description |
    |---|---|
    | **Visit duration** | How long to stay on each site (seconds) |
    | **Pages per site** | Number of pages to browse on each domain |
    | **Delay between sites** | Pause between site visits (seconds) |
    | **Scroll behavior** | Simulate scrolling on pages |

    {/* Screenshot: Cookie Robot parameters form */}
  </Step>
  <Step title="Start">
    Click **Start** to begin the warm-up process. Cookie Robot runs in the background — you can monitor progress in the task panel.

    {/* Screenshot: Cookie Robot progress panel */}
  </Step>
</Steps>

<Note>
  Cookie Robot uses the proxy assigned to each profile. Make sure proxies are configured and working before starting a warm-up session.
</Note>

## Best practices

- Run Cookie Robot on new profiles **before** using them on target platforms
- Use **2–5 second** visit durations for realistic behavior
- Include a mix of **search engines, news sites, and social media** in custom lists
- Space warm-up sessions over **multiple days** for best results
