## What is a browser profile

A browser profile is an isolated browser instance with its own unique digital fingerprint, cookies, local storage, and browsing history. Each profile behaves as a completely separate browser — websites cannot link profiles to each other or to your real identity.

<Frame><img src="/images/atmos_ui/browser/atmos_browser_profiles.png" alt="Browser profiles list" /></Frame>

## Fingerprint components

Every profile generates a unique combination of browser parameters:

| Parameter | Description |
|---|---|
| **User-Agent** | Browser (Chrome version) and OS identification string |
| **Screen resolution** | Display dimensions |
| **Platform** | OS platform identifier (e.g. Windows 10.0) |
| **Language** | Browser language and locale |
| **Timezone** | Geographic timezone setting |
| **Geolocation** | Based on IP address |
| **DNS** | Based on IP address |
| **WebRTC** | Leak protection settings (based on IP address) |
| **Canvas** | Canvas rendering noise |
| **Audio** | AudioContext noise |
| **Rectangles** | DOM rectangles noise |
| **WebGL** | GPU renderer noise |

<Info>
  Fingerprints are generated automatically when you create a profile. Click **New fingerprint** to regenerate, or adjust individual parameters manually.
</Info>

## Profile states

| State | Description |
|---|---|
| **Running** | Browser window is currently open (green indicator) |
| **Not used** | Browser is closed, data saved (gray indicator) |

## Creating a profile

<Steps>
  <Step title="Click New Profile">
    Click the **New Profile** button in the top-right corner.
  </Step>
  <Step title="Configure fingerprint">
    Set the profile name, tags, platform (e.g. Windows), and Chrome version. Enable **Keep up to date** to auto-update the Chrome version. The preview panel on the right shows all generated fingerprint parameters.

    <Frame><img src="/images/atmos_ui/browser/atmos_browser_new_profile.png" alt="Create new profile with fingerprint preview" /></Frame>
  </Step>
  <Step title="Assign a proxy (optional)">
    Enter proxy details in the proxy fields. Supported types: **HTTP** and **SOCKS5**.

    Proxy format:

    ```text
    protocol://host:port:user:pass
    protocol://user:pass@host:port
    ```
  </Step>
  <Step title="Save">
    Click **Save profile**. Click the profile row to launch the browser.
  </Step>
</Steps>

## Editing a profile

You can fully edit a profile after creation with two constraints:

- **Chrome version cannot be downgraded** — if a profile was created with Chrome 144, only 145+ is available
- **Primary wallet is permanent** — a wallet assigned at creation becomes the primary wallet and cannot be unlinked. Secondary wallets can be added or removed later.

## Proxy settings

Each profile has its own proxy. Supported types: **HTTP** and **SOCKS5**.

To change a profile's proxy, open the profile editor and update the proxy fields.

<Warning>
  Running profiles without a proxy exposes your real IP address. Always assign a proxy for production use.
</Warning>
