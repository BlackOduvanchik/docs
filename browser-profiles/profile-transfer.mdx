## Overview

Profile transfer lets you send browser profiles to another Oxygen Atmos user. The recipient gets a full copy of the profile including fingerprint, cookies, and browsing data. This is useful for teamwork, selling warmed accounts, or moving profiles between your own accounts.

## Send a profile

<Steps>
  <Step title="Select profiles">
    In the profile list, select one or more profiles using the checkboxes.

    {/* Screenshot: selected profiles with checkboxes */}
  </Step>
  <Step title="Click Transfer">
    Click the **Transfer** button in the action bar or right-click and select **Transfer**.

    {/* Screenshot: transfer button in action bar */}
  </Step>
  <Step title="Enter recipient">
    Enter the recipient's Oxygen Atmos username or email address.

    {/* Screenshot: transfer dialog with recipient field */}
  </Step>
  <Step title="Confirm transfer">
    Review the transfer details and click **Send**. The profiles will appear in the recipient's [Received Profiles](/folders/received-profiles-folder) folder.
  </Step>
</Steps>

## Receive a profile

Transferred profiles appear in your **Received Profiles** folder. From there you can:

- Move them to any folder
- Edit fingerprint and proxy settings
- Launch them immediately

<Info>
  Transferred profiles retain their original fingerprint and cookies. The sender loses access to the profile after transfer.
</Info>

## Transfer options

| Option | Description |
|---|---|
| **Include proxy** | Transfer the proxy assignment with the profile |
| **Include cookies** | Transfer saved cookies and session data |
| **Include notes** | Transfer profile notes and tags |

<Warning>
  Transferred profiles count against the recipient's profile quota. Make sure the recipient has enough available slots.
</Warning>
