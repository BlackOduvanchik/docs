## Edit a proxy

<Steps>
  <Step title="Open the proxy">
    In the proxy manager, click on the proxy name or select **Edit** from the context menu.

    {/* Screenshot: proxy edit button */}
  </Step>
  <Step title="Modify settings">
    Update any fields: name, type, host, port, username, or password.

    {/* Screenshot: proxy edit form */}
  </Step>
  <Step title="Re-check connection">
    Click **Check Proxy** to verify the updated proxy works.
  </Step>
  <Step title="Save">
    Click **Save** to apply changes.
  </Step>
</Steps>

<Info>
  Editing a proxy updates it for all profiles that use it. You don't need to reassign the proxy to each profile.
</Info>

## Change proxy for a profile

To swap the proxy assigned to a profile:

1. Open the profile editor
2. In the **Proxy** section, click **Change**
3. Select a different proxy from the list or add a new one
4. Save the profile

{/* Screenshot: proxy selector in profile editor */}

## Bulk edit

Select multiple proxies in the proxy manager to perform bulk operations:

- **Change type** — switch all selected proxies to a different type
- **Re-check** — verify connectivity for all selected proxies
- **Delete** — remove all selected proxies
