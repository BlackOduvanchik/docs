## Delete a single proxy

1. In the proxy manager, right-click the proxy
2. Select **Delete**
3. Confirm the deletion

{/* Screenshot: proxy delete confirmation */}

## Bulk delete

1. Select multiple proxies using checkboxes
2. Click **Delete** in the action bar
3. Confirm the deletion

## What happens to assigned profiles

When you delete a proxy that's assigned to profiles:

- The profile's proxy assignment is **cleared** (set to "No proxy")
- The profile itself is **not deleted**
- The profile will use your direct connection until you assign a new proxy

<Warning>
  Proxy deletion is immediate and irreversible. Make sure no active profiles depend on the proxy before deleting.
</Warning>

<Note>
  To temporarily disable a proxy without deleting it, you can remove it from profiles individually instead of deleting the proxy entry.
</Note>
