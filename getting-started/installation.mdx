## Availability

Oxygen Atmos is currently available for **Windows**. macOS support is in development.

## System requirements

| Requirement | Minimum | Recommended |
|---|---|---|
| **OS** | Windows 10 | Windows 11 |
| **RAM** | 4 GB | 8 GB+ |
| **Disk** | 2 GB free space | SSD with 10 GB+ |
| **CPU** | Dual-core | Quad-core+ |

## Download

<Steps>
  <Step title="Download the installer">
    Go to [wallet.oxygen.tools/en/download](https://wallet.oxygen.tools/en/download) and download the installer for Windows.
  </Step>
  <Step title="Run the installer">
    Run the `.exe` installer and follow the on-screen instructions.
  </Step>
  <Step title="Launch the application">
    Open Oxygen Atmos to begin the first-time setup.
  </Step>
</Steps>

## First-time setup

When you launch Oxygen Atmos for the first time, you go through a two-step setup:

<Steps>
  <Step title="Enter your API key">
    Paste the API key from your account on [wallet.oxygen.tools](https://wallet.oxygen.tools). This activates the application.

    <Frame><img src="/images/atmos_ui/onboarding/atmos_first_time_setup_1.png" alt="First time setup — API key entry" /></Frame>

  </Step>
  <Step title="Set a master password">
    Create a master password to encrypt all sensitive data (private keys, API keys, settings) stored locally on your machine. You can also choose **Continue without encryption**, but this is not recommended.
  </Step>
</Steps>

<Info>
  All data is stored locally — Oxygen Atmos does not send your keys or credentials to external servers.
</Info>

## Auto-updates

When a new version is available, you'll see an in-app notification. To update, restart Oxygen Atmos.
