## Welcome to Oxygen Atmos

Oxygen Atmos is a desktop application that combines a Chromium-based antidetect browser with full web3 wallet management. Each browser profile operates as an isolated environment with a unique digital fingerprint, while built-in modules handle wallet management, exchange accounts, and cryptocurrency transfers at scale.

## Use cases

| Use case | Description |
|---|---|
| **Airdrop farming** | Run 50–500+ wallets across protocols with isolated fingerprints, proxies, and session persistence |
| **DeFi & trading** | Isolated profiles for DEXs, lending, perps, and bridges — each with its own wallet, proxy, and session. Access geo-restricted frontends per-profile without affecting other sessions |
| **Wallet management** | Track activity across wallets with linked browser profiles, tags, and activity notes |
| **Team operations** | Coordinate farming across a team with PIN-protected profiles, tags, and organized groups |
