## Prerequisites

- An active Dolphin Anty account with profiles to export
- An Oxygen Atmos account with sufficient profile slots
- Both applications installed on your machine

## Migration steps

<Steps>
  <Step title="Export profiles from Dolphin Anty">
    In Dolphin Anty, select the profiles you want to migrate and export them.
  </Step>
  <Step title="Import into Oxygen Atmos">
    In Oxygen Atmos, open the migration import tool and upload the Dolphin Anty export file. Oxygen Atmos will parse the file and show a preview of profiles to import.
  </Step>
  <Step title="Configure import options">
    Choose how to handle proxy assignments:

    - **Import proxies** — re-create proxy entries in Oxygen Atmos
    - **Overwrite duplicates** — replace profiles with matching names
  </Step>
  <Step title="Start import">
    Click **Import** and wait for the process to complete. You'll see a summary of imported profiles.
  </Step>
</Steps>

## What gets imported

| Data | Imported | Notes |
|---|---|---|
| Fingerprint config | Yes | OS, browser, screen, WebGL, Canvas, Audio |
| Cookies | Yes | Transferred as-is |
| Proxy settings | Yes | Requires valid proxy credentials |
| Profile notes | Yes | — |
| Tags | Yes | Mapped to Oxygen Atmos tags |
| Extensions | No | Must be added manually in Settings → Browser |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Import file is not recognized">
    Make sure the file was exported from Dolphin Anty in a supported format.
  </Accordion>
  <Accordion title="Proxies not working after import">
    Verify that your proxy credentials are still valid. Imported proxies use the same host, port, and authentication from Dolphin Anty — if they've expired, update them in the profile editor.
  </Accordion>
  <Accordion title="Missing cookies or sessions">
    Cookie import depends on the export completeness from Dolphin Anty. If sessions are lost, re-warm profiles by visiting the target websites manually.
  </Accordion>
</AccordionGroup>
