## Overview

Oxygen Atmos supports importing profiles from other antidetect browsers. Migration preserves your fingerprint configurations, cookies, and proxy assignments so you can switch without starting over.

## Supported browsers

| Browser | Status | Notes |
|---|---|---|
| **Dolphin Anty** | Supported | Full profile and proxy import |
| **Other browsers** | Coming soon | Export as JSON and import manually |

## What gets migrated

- Browser profile fingerprint settings
- Saved cookies and local storage
- Proxy assignments
- Profile notes and tags
- Folder structure (where supported)

<Note>
  Extensions and bookmarks are not included in migration. You'll need to reinstall extensions manually after importing profiles.
</Note>

## Before you start

1. **Export your data** from the source browser (see the specific migration guide)
2. **Back up your profiles** in the source browser before starting
3. **Ensure you have enough profile slots** in your Oxygen Atmos subscription
