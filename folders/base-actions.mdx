## Rename a folder

1. Right-click the folder in the sidebar
2. Select **Rename**
3. Enter the new name and press **Enter**

{/* Screenshot: folder rename context menu */}

## Reorder folders

Drag folders in the sidebar to reorder them. The order is saved automatically and synced across devices.

## Delete a folder

<Steps>
  <Step title="Right-click the folder">
    Select **Delete Folder** from the context menu.

    {/* Screenshot: delete folder context menu */}
  </Step>
  <Step title="Choose what happens to profiles">
    - **Move to default folder** — profiles are moved out of the folder before deletion
    - **Delete profiles** — profiles inside are moved to the Trash Bin

    {/* Screenshot: folder deletion confirmation dialog */}
  </Step>
  <Step title="Confirm">
    Click **Delete** to confirm. The folder is removed immediately.
  </Step>
</Steps>

<Warning>
  Deleting a folder is irreversible. Profiles moved to the Trash Bin can be restored within 30 days.
</Warning>

## Folder context menu

Right-click any folder to access:

| Action | Description |
|---|---|
| **Rename** | Change the folder name |
| **Delete** | Remove the folder |
| **Start all** | Launch all profiles in the folder |
| **Stop all** | Close all running profiles in the folder |
| **Select all** | Select all profiles in the folder |
