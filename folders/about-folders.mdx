## Overview

Folders help you organize browser profiles into logical groups. Use folders to separate profiles by project, client, platform, or any other criteria that fits your workflow.

{/* Screenshot: folder list in the sidebar */}

## Folder structure

Oxygen Atmos uses a flat folder structure:

- Each profile belongs to **one folder** at a time
- The **default folder** holds profiles not assigned to any folder
- Special folders: **Received Profiles** and **Trash Bin**

## Default folders

| Folder | Description |
|---|---|
| **All Profiles** | View of all profiles across all folders |
| **Received Profiles** | Profiles received via [transfer](/browser-profiles/profile-transfer) |
| **Trash Bin** | Deleted profiles awaiting permanent removal |

## Creating a folder

<Steps>
  <Step title="Click New Folder">
    In the sidebar, click the **+** icon next to **Folders** or click **New Folder**.

    {/* Screenshot: new folder button in sidebar */}
  </Step>
  <Step title="Name the folder">
    Enter a descriptive name for the folder (e.g., "Facebook Accounts", "Client A").
  </Step>
  <Step title="Done">
    The folder appears in the sidebar. Drag profiles into it or use the move action.
  </Step>
</Steps>

## Moving profiles between folders

- **Drag and drop** — drag profiles from the list onto a folder in the sidebar
- **Context menu** — right-click a profile → **Move to Folder** → select destination
- **Bulk move** — select multiple profiles with checkboxes, then use **Move to Folder**

<Info>
  Moving a profile between folders does not affect its fingerprint, cookies, or proxy settings.
</Info>
