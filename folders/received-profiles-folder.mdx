## Overview

The **Received Profiles** folder is a special folder that holds profiles sent to you by other Oxygen Atmos users via [profile transfer](/browser-profiles/profile-transfer). It acts as an inbox for incoming profiles.

{/* Screenshot: Received Profiles folder in the sidebar */}

## How it works

1. Another user sends you a profile via the transfer feature
2. The profile appears in your **Received Profiles** folder
3. You receive a notification about the incoming transfer
4. You can then move the profile to any regular folder

## Managing received profiles

When a profile arrives, you can:

- **Accept and move** — move it to a regular folder to start using it
- **Edit** — change fingerprint, proxy, or other settings before use
- **Launch** — start the browser directly from the Received Profiles folder
- **Delete** — remove the profile if you don't need it

<Info>
  Received profiles count against your profile quota. If you're at your limit, you won't be able to receive new transfers until you free up slots.
</Info>

## Best practices

- Review received profiles promptly — don't leave them in the inbox
- Check proxy assignments after receiving — the sender's proxies may not transfer
- Rename profiles to match your naming convention after accepting
