## Filter by tags

Show only wallets with specific tags:

<Steps>
  <Step title="Open tag filter">
    Click **Tags** above the wallet list.
  </Step>
  <Step title="Select tags">
    Choose one or more tags in the dropdown. Only wallets with the selected tags will appear.

    <Frame caption="Tag filter dropdown">
      <img src="/images/placeholder.png" />
    </Frame>
  </Step>
  <Step title="Reset">
    To clear the filter, click **Reset** in the tag dropdown.
  </Step>
</Steps>

## Search by alias

Find wallets by their display name:

1. Click **Search** above the wallet list
2. Type the alias you're looking for
3. The list filters in real time to show matching wallets

<Frame caption="Alias search">
  <img src="/images/placeholder.png" />
</Frame>

## Sorting

Sort the wallet list by clicking on any column header:

| Column | Sort by |
|---|---|
| **Type** | Blockchain type (alphabetical) |
| **Alias** | Wallet display name (alphabetical) |
| **Balance** | Token balance (ascending/descending) |

Click a column header once for ascending order, click again for descending.
