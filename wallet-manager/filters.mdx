## Quick filters

The toolbar above the wallet list provides quick filtering:

- **Tags** — click **Tags** to filter by one or more tags. Click **Reset** to clear.
- **Search** — type a wallet alias or address to filter the list in real time.

## Balance filters

Click the **filter icon** to open the filter dialog with these options:

| Filter | Description |
|---|---|
| **Don't show assets worth less than** | Hide tokens below a dollar threshold |
| **Show only verified assets** | Toggle to exclude unverified tokens |
| **Show maximum number of chains** | Limit how many chains are displayed per wallet |
| **Show maximum number of protocols** | Limit protocols shown per wallet |
| **Show maximum number of assets** | Limit individual assets shown per wallet |

## Sorting

Sort the wallet list by clicking on any column header:

| Column | Sort by |
|---|---|
| **Alias** | Wallet display name (alphabetical) |
| **Balance** | Token balance (ascending/descending) |

Click a column header once for ascending order, click again for descending.
