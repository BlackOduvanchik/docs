## Edit a single wallet

You can change the withdrawal address, alias, and tags for any imported wallet.

<Steps>
  <Step title="Open the edit menu">
    Click the **edit icon** next to the wallet you want to modify.

    <Frame caption="Wallet edit icon">
      <img src="/images/placeholder.png" />
    </Frame>
  </Step>
  <Step title="Update fields">
    Change the **withdrawal address**, **alias**, or **tags** as needed.
  </Step>
  <Step title="Save">
    Click **Save Changes**.
  </Step>
</Steps>

## Edit multiple wallets

To bulk-update the withdrawal address or alias for multiple wallets, re-import them using any of the [import methods](/wallet-manager/import-wallets) with the updated data.

<Note>
  Re-importing a wallet with the same private key or address **updates the existing entry** — it does not create a duplicate.
</Note>
