## Python (preview)

```python
from oxygen_atmos import OxygenAtmos

client = OxygenAtmos(api_key="YOUR_API_KEY")

# List profiles
profiles = client.profiles.list()

# Start a profile
session = client.profiles.start("prof_abc123")
print(session.ws_endpoint)

# Stop a profile
client.profiles.stop("prof_abc123")
```

## JavaScript/TypeScript (preview)

```typescript
import { OxygenAtmos } from '@oxygen-atmos/sdk';

const client = new OxygenAtmos({ apiKey: 'YOUR_API_KEY' });

// List profiles
const profiles = await client.profiles.list();

// Start a profile
const session = await client.profiles.start('prof_abc123');
console.log(session.wsEndpoint);

// Stop a profile
await client.profiles.stop('prof_abc123');
```
