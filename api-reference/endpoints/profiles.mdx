</Note>

## List profiles

```
GET /api/v1/profiles
```

Returns all browser profiles in your account.

**Query parameters**

| Parameter | Type | Description |
|---|---|---|
| `folder_id` | string | Filter by folder ID |
| `status` | string | Filter by status: `new`, `running`, `stopped` |
| `tag` | string | Filter by tag name |
| `limit` | integer | Max results (default: 50, max: 200) |
| `offset` | integer | Pagination offset |

**Response**

```json
{
  "success": true,
  "data": [
    {
      "id": "prof_abc123",
      "name": "Profile 1",
      "status": "stopped",
      "os": "windows",
      "browser": "chrome",
      "folder_id": "fold_xyz",
      "proxy_id": "prx_456",
      "tags": ["facebook", "farm-1"],
      "created_at": "2024-01-15T10:30:00Z",
      "updated_at": "2024-01-20T14:00:00Z"
    }
  ],
  "total": 150,
  "limit": 50,
  "offset": 0
}
```

## Get profile

```
GET /api/v1/profiles/{profile_id}
```

Returns detailed information about a single profile including fingerprint configuration.

**Response**

```json
{
  "success": true,
  "data": {
    "id": "prof_abc123",
    "name": "Profile 1",
    "status": "stopped",
    "os": "windows",
    "browser": "chrome",
```

## Create profile

```
POST /api/v1/profiles
```

**Request body**

```json
{
  "name": "New Profile",
  "os": "windows",
  "browser": "chrome",
  "proxy_id": "prx_456",
  "folder_id": "fold_xyz",
  "tags": ["campaign-1"],
  "notes": "Created via API"
}
```

**Response** — `201 Created`

```json
{
  "success": true,
  "data": {
    "id": "prof_new789",
    "name": "New Profile",
    "status": "new",
    "fingerprint": { "..." : "auto-generated" }
  }
}
```

---

## Update profile

```
PATCH /api/v1/profiles/{profile_id}
```

**Request body** — only include fields to update:

```json
{
  "name": "Updated Name",
  "proxy_id": "prx_789",
  "tags": ["updated-tag"]
}
```

---

## Delete profile

```
DELETE /api/v1/profiles/{profile_id}
```

Moves the profile to the Trash Bin. Returns `204 No Content` on success.

<Warning>
  To permanently delete, use `DELETE /api/v1/profiles/{profile_id}?permanent=true`.
</Warning>
