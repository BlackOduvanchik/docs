## Oxygen Atmos API

The Oxygen Atmos API lets you automate browser profile management, proxy configuration, and browser lifecycle operations. Use it to integrate Oxygen Atmos into your automation pipelines, build custom tools, or manage profiles programmatically.

<Note>
  The API is currently in development. This documentation describes the planned endpoints and will be updated as the API becomes available.
</Note>

## Base URL

```
http://localhost:3001/api/v1
```

The API runs locally alongside the Oxygen Atmos desktop application.
