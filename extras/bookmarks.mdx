## Overview

Bookmarks in Oxygen Atmos can be managed at two levels:

- **Per-profile bookmarks** — saved inside each individual profile
- **Shared bookmarks** — a common set applied to all or selected profiles

## Per-profile bookmarks

Bookmarks you create while browsing inside a profile are saved to that profile only. They persist across sessions and sync with the [Synchronizer](/browser-profiles/synchronizer).

## Shared bookmarks

Shared bookmarks let you define a common bookmark set that's loaded into multiple profiles:

<Steps>
  <Step title="Open bookmarks manager">
    Go to **Extras** → **Bookmarks**.

    {/* Screenshot: bookmarks manager */}
  </Step>
  <Step title="Create a bookmark set">
    Click **New Set** and give it a name (e.g., "Social Media Links").
  </Step>
  <Step title="Add bookmarks">
    Add URLs with labels to the set.

    ```text
    Facebook | https://facebook.com
    Google Ads | https://ads.google.com
    Amazon Seller | https://sellercentral.amazon.com
    ```
  </Step>
  <Step title="Assign to profiles">
    Select which profiles or folders should receive this bookmark set.
  </Step>
</Steps>

<Info>
  Shared bookmarks are read-only inside the profile browser. To edit them, use the bookmarks manager in the Oxygen Atmos dashboard.
</Info>
