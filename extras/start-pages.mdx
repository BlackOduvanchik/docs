## Overview

Start pages define which URLs open in new tabs when a browser profile launches. Use start pages to jump straight into the platforms you work with.

## Configure start pages

<Steps>
  <Step title="Open profile editor">
    Click **Edit** on a profile in the dashboard.

    {/* Screenshot: profile edit button */}
  </Step>
  <Step title="Go to Start Pages tab">
    Click the **Start Pages** section in the profile editor.

    {/* Screenshot: start pages section */}
  </Step>
  <Step title="Add URLs">
    Enter one or more URLs to open on launch:

    ```text
    https://facebook.com
    https://ads.google.com
    https://sellercentral.amazon.com
    ```
  </Step>
  <Step title="Save">
    Click **Save**. The URLs will open as tabs next time you start the profile.
  </Step>
</Steps>

## Global start pages

To set start pages for all profiles at once:

1. Go to **Extras** → **Start Pages**
2. Add URLs to the global list
3. These pages open in addition to per-profile start pages

{/* Screenshot: global start pages settings */}

<Note>
  If a profile has both global and per-profile start pages, all URLs open together. Duplicates are opened only once.
</Note>
