## Overview

Tags in Oxygen Atmos are **shared across wallets and browser profiles**. A tag created in the wallet manager is available in browser profiles and vice versa. This lets you maintain consistent grouping — for example, tag wallets and their linked profiles with the same campaign label.

</Frame>

## Create and assign tags

<Steps>
  <Step title="Select items">
    Check the wallets or profiles you want to tag.
  </Step>
  <Step title="Open the tag editor">
    Click **Edit Tags** in the top-right corner.
  </Step>
  <Step title="Create a new tag">
    Click **Create Tag**, enter a **name** and choose a **color** (or leave it as **Random**).
  </Step>
  <Step title="Apply">
    Click **Add Tag**, then click **Save**.

    <Frame caption="Tag creation dialog">
      <img src="/images/placeholder.png" />
    </Frame>
  </Step>
</Steps>

<Note>
  You can apply multiple tags to a single wallet or profile. Tags work the same way in both the wallet manager and the profile list.
</Note>

## Edit tags

<Steps>
  <Step title="Open settings">
    Go to **Settings** in the left navigation panel.
  </Step>
  <Step title="Navigate to tags">
    Click **Tags** in the top-right corner.
  </Step>
  <Step title="Edit">
    Click the **edit icon** next to the tag you want to modify. Change the **name** and/or **color**, then click **Edit Tag**.
  </Step>
</Steps>

<Note>
  Tag changes apply automatically everywhere the tag is used — both wallets and profiles.
</Note>

## Delete tags

<Steps>
  <Step title="Open settings">
    Go to **Settings** in the left navigation panel.
  </Step>
  <Step title="Navigate to tags">
    Click **Tags** in the top-right corner.
  </Step>
  <Step title="Select and delete">
    Check one or more tags, then click **Delete**.
  </Step>
</Steps>

<Warning>
  Deleted tags are automatically removed from all wallets and profiles where they were assigned.
</Warning>

## Filter by tags

You can filter both wallets and profiles by tag:

1. Click **Tags** above the list
2. Select one or more tags — only items with those tags will appear
3. Click **Reset** to clear the filter
